准备工作

  • 安装pipenv pip install pipenv
  • 虚拟环境目录： C:\Users\username\.virtualenvs\project-xxxxxxxx
  • 激活项目虚拟环境，执行： pipenv shell
  • 安装Flask，pipenv install flask

常用pipenv命令

  • 退出虚拟环境exit
  • 进入虚拟环境pipenv shell，如不存在虚拟环境可自动创建
  • 卸载包pipenv uninstall flask
  • 查看包之间的依赖关系pipenv graph
  • 查看虚拟环境路径pipenv --venv
  • 创建虚拟环境pipenv
  • 查看所有安装包pip list

设置项目的虚拟环境

  • 进入项目目录，依次使用如下命令pipenv shell $\Rightarrow$ pipenv --venv即可得到虚拟环境目录
